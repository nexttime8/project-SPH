## 讲项目提交到本地仓库

1. git init
   1. 创建本地仓库
   2. 只有第一次需要执行
2. git add .
   1. 开始跟踪新文件、已跟踪已修改放到暂存区，有冲突文件标记为已解决状态
   2. 将所有新增修改的文件，放入到暂存区
3. git status
   1. 如果显示全绿色 A，表示已经添加到本地项目暂存区，就可以 commit 了
4. git commit -m "description"
   1. 提交到本地仓库

## 将本地 Git 仓库中的提交推送到远程 Git 仓库流程

1. git push -u origin main
   1. 同步到远程仓库
   2. 关掉科学上网~
